Zastosowanie C++

C++ ma zastosowanie tam, gdzie wymagana jest wysoka wydajność, odpowiednie zarządzanie pamięcią i sprzętem oraz uruchamianie aplikacji

Skoro C++ jest niskopoziomowy, nietrudno się domyśleć, że znalazł zastosowanie przy systemach operacyjnych. Spora część z nich została napisana w tym języku. Dla uściśnienia, jądra tych systemów są napisane w C lub w Assemlerze, jednak niektóre moduły, aplikacje lub sterowniki to przed wszystkim C++. Te systemu to np.:

  • Apple OS X
  • IBM OS/400
  • Windows NT Vista
  • Windows XP
  • Windows Vista
  • Windows 7

W tym języki są napisane popularne silniki graficzne takie jak Unity, Unreal Engine czy CryEngine. Dodatkowo z zalet wydajnosci tego języka korzystają aplikacjie do obróbki zdjęć, programy biurowe czy aplikacjie dużych korporacji, takie jak.:

  • produkty Adobe
  • Pakiet biurowy Office
  • Chrome
  • Blender
  • Serwis Youtube.com
